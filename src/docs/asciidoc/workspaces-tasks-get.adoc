== 작업 상세 조회

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``GET``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |특정 작업의 상세 정보를 조회합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|응답 범위 |작업 본문, 담당자, 태그, 진행도, 생성자와 생성/수정 시각, 댓글 첫 페이지를 반환합니다.
|댓글 |``commentPage``에는 생성 시각이 오래된 순서의 댓글 첫 페이지가 최대 20개까지 포함됩니다. 이후 페이지는 작업 댓글 목록 조회 API를 호출합니다.
|응답 |``200 OK``
|===
