== 현재 사용자 워크스페이스 목록 조회

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``GET``
|경로 |``/v1/users/me/workspaces``
|API 상태 |[.badge.stability.stable]#Stable#
|목적 |현재 사용자가 가입한 워크스페이스 목록을 조회합니다.
|인증 |JWT access token이 필요합니다.
|조회 기준 |워크스페이스 본체 전체 조회가 아니라 현재 사용자에게 연결된 활성 워크스페이스 멤버십을 기준으로 조회합니다.
|응답 형식 |배열 응답 공통 규칙에 따라 ``data.items``와 ``data.totalCount``를 반환합니다.
|===
