== 워크스페이스 멤버 탈퇴

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``DELETE``
|경로 |``/v1/workspaces/{workspaceId}/members/withdraw``
|API 상태 |[.badge.stability.stable]#Stable#
|목적 |현재 인증된 사용자가 자신이 가입된 워크스페이스에서 스스로 탈퇴합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청 사용자는 해당 워크스페이스의 활성 멤버여야 합니다.
|OWNER 탈퇴 |OWNER가 탈퇴할 때 활성 ADMIN이 존재하면 ADMIN으로 지정된 지 가장 오래된 멤버를 OWNER로 승격한 뒤 탈퇴합니다. 활성 ADMIN이 없으면 탈퇴를 거부합니다.
|처리 방식 |요청 사용자의 워크스페이스 멤버십을 soft delete 처리하고, 탈퇴 이력을 강제 퇴장 이력과 별도 테이블에 저장합니다.
|성공 응답 |``200 OK``와 빈 성공 응답 본문을 반환합니다.
|===
