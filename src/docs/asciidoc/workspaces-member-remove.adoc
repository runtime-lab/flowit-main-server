== 워크스페이스 멤버 강제 퇴장

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |`DELETE`
|경로 |`/v1/workspaces/{workspaceId}/members/{memberId}`
|API 상태 |[.badge.stability.stable]#Stable#
|목적 |워크스페이스 OWNER 또는 ADMIN이 대상 멤버를 강제 퇴장시킵니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 `OWNER` 또는 `ADMIN`이어야 합니다.
|대상 제한 |`OWNER`는 강제 퇴장 대상이 될 수 없습니다. 요청자 자신도 강제 퇴장시킬 수 없습니다.
|처리 방식 |대상 워크스페이스 멤버십을 soft delete 처리합니다.
|성공 응답 |`200 OK`와 빈 성공 응답 본문을 반환합니다.
|===
