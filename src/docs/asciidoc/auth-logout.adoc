== 로그아웃

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |`POST`
|경로 |`/v1/public/auth/logout`
|목적 |Refresh token 세션을 종료하고 브라우저 쿠키를 만료 처리합니다.
|요청 본문 |없습니다.
|쿠키 |`flowit_refresh_token` 쿠키가 있으면 해당 refresh token을 로그아웃 처리합니다.
|===

== 클라이언트 주의사항

브라우저 기반 클라이언트는 쿠키가 함께 전송되도록 credentials 옵션을 켭니다.

[source,javascript]
----
await fetch("/v1/public/auth/logout", {
  method: "POST",
  credentials: "include"
});
----

Axios를 사용하는 경우 `withCredentials`를 사용합니다.

[source,javascript]
----
await axios.post("/v1/public/auth/logout", null, {
  withCredentials: true
});
----

프론트엔드와 API의 origin이 다르면 브라우저의 credentials/CORS 정책을 따릅니다. 클라이언트는 위 credentials 옵션을 사용해야 하며, wildcard origin(`*`)과 credentials 요청 조합은 브라우저에서 허용되지 않습니다.
