== 개요

API 응답에서 문자열로 전달되는 enum 값의 의미를 도메인별 문서에서 관리합니다.

이 페이지는 전체 enum의 색인입니다. 실제 enum 값과 설명은 각 enum을 소유한 도메인 문서에만 둡니다. 다른 도메인에서 같은 값을 참조해야 하는 경우 값을 복제하지 않고 원본 도메인 문서로 연결합니다.

== 운영 기준

[cols="1,4",role="endpoint-summary"]
|===
|소유 기준 |enum은 값을 정의하거나 응답 계약의 의미를 결정하는 도메인 문서에 둡니다.
|참조 기준 |다른 도메인이 같은 값을 사용하더라도 설명을 복제하지 않고 소유 도메인 문서로 링크합니다.
|링크 기준 |API 필드 설명은 가능하면 이 색인보다 실제 enum 섹션으로 직접 연결합니다.
|확장 기준 |새 enum이 추가되면 해당 도메인 enum 문서에 먼저 추가하고 이 색인에는 도메인 문서 링크만 보강합니다.
|===
