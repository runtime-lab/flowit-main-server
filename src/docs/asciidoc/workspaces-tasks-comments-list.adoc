== 작업 댓글 목록 조회

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``GET``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}/comments``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |특정 작업에 달린 댓글 목록을 조회합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|정렬 |생성 시각이 오래된 댓글부터 반환합니다. 동일 시각이면 식별자 오름차순입니다.
|페이지 |``size``를 생략하면 기본 20개를 반환합니다.
|수정 여부 |생성 이후 내용 수정이 발생한 댓글은 ``edited``가 ``true``입니다.
|응답 |``200 OK``
|===
