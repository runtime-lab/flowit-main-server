== 작업 상태 변경

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``PATCH``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}/status``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |작업 상태만 별도로 변경합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|상태 |``TODO``, ``IN_PROGRESS``, ``DONE`` 간 자유 전이를 허용합니다.
|처리 방식 |상태 변경 이력을 별도로 남깁니다.
|정렬 |보드 안의 작업 순서 변경은 이 API에서 처리하지 않습니다. 정렬 정책은 별도 API로 확장합니다.
|성공 응답 |``200 OK``와 빈 성공 응답 본문을 반환합니다.
|===
