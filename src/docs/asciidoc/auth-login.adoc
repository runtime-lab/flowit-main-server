== Refresh Token Cookie

로그인 성공 응답은 `Set-Cookie` 헤더로 `flowit_refresh_token` 쿠키를 전달합니다. 이 쿠키는 JavaScript에서 읽을 수 없는 `HttpOnly` 쿠키이며, 클라이언트는 refresh token 값을 직접 저장하거나 요청 본문에 담지 않습니다.

[source,http]
----
Set-Cookie: flowit_refresh_token=...; HttpOnly
----

쿠키 속성은 요청 환경과 브라우저 정책에 따라 다를 수 있습니다. 클라이언트는 쿠키 값을 직접 읽거나 속성 값을 고정 가정하지 않고, 브라우저의 쿠키 처리에 맡깁니다.
