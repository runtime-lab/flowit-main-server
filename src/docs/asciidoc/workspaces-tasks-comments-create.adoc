== 작업 댓글 생성

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``POST``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}/comments``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |특정 작업에 댓글을 생성합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|본문 |``contentMarkdown`` 원문을 저장합니다. 서버는 Markdown을 렌더링하지 않습니다. 총 길이 1000자 제한 markdown은 추후 확장성을 고려한 임시적인 필드명 입니다.
|최근 활동 |댓글 작성은 현재 워크스페이스 최근 활동에 기록하지 않습니다.
|응답 |``201 Created``
|===
