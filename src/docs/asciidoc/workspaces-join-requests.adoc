== 가입 요청 이력 조회

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |`GET`
|경로 |`/v1/workspaces/{workspaceId}/join-requests`
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |워크스페이스 가입 요청과 각 요청의 상태 변경 이력을 조회합니다.
|인증 |JWT access token이 필요합니다.
|권한 |워크스페이스 `OWNER` 또는 `ADMIN`만 가입 요청 이력을 조회할 수 있습니다.
|이력 |각 이력 행에는 이전 상태, 다음 상태, 전이 이벤트, 전이를 발생시킨 사용자, 전이 시각이 포함됩니다.
|===
