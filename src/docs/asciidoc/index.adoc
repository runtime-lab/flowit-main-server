|link:auth-token-refresh.html[토큰 재발급 API]

|link:auth-logout.html[로그아웃 API]

[%collapsible]
.유저 API
====

[cols="28,30,42",role="api-index"]
|===
|문서 |Endpoint |설명

|link:users-join.html[회원가입 API]
|`POST /v1/public/users/join`
|회원가입 API 요청/응답과 중복 이메일 에러 응답을 설명합니다.

|link:users-me.html[현재 사용자 조회 API]
|`GET /v1/users/me`
|인증된 사용자의 기본 정보, 프로필 이미지 조회 URL, 소속 워크스페이스 목록을 설명합니다.

|link:users-me-workspaces.html[현재 사용자 워크스페이스 목록 API]
|`GET /v1/users/me/workspaces`
|현재 사용자가 가입한 워크스페이스 목록을 별도 목록 응답으로 조회하는 API를 설명합니다.

|link:users-me-update.html[현재 사용자 수정 API]
|`PATCH /v1/users/me`
|사용자 ID를 요청에서 받지 않고 인증 토큰 기준으로 현재 사용자의 수정 가능한 정보를 변경하는 API를 설명합니다.

|link:users-password.html[현재 사용자 비밀번호 변경 API]
|`PATCH /v1/users/me/password`
|현재 비밀번호 검증, `token_version` 증가, refresh token 쿠키 만료 처리를 포함한 비밀번호 변경 API를 설명합니다.

|link:users-profile-image.html[유저 프로필 이미지 API]
a|`PUT /v1/users/me/profile-image` +
`GET /v1/users/me/profile-image`
|인증된 사용자의 프로필 이미지 교체와 이미지 바이너리 조회 방식을 설명합니다.
|===

[%collapsible]
.워크스페이스 API
====

[cols="28,30,42",role="api-index"]
|===
|문서 |Endpoint |설명

|link:workspaces-create.html[워크스페이스 생성 API]
|`POST /v1/workspaces`
|인증된 사용자가 워크스페이스를 생성하는 요청/응답과 생성 리소스 위치를 설명합니다.

|link:workspaces-get.html[워크스페이스 조회 API]
|`GET /v1/workspaces/{workspaceId}`
|워크스페이스 활성 멤버가 워크스페이스 자체 정보를 조회하는 요청/응답과 권한 제한을 설명합니다.

|link:workspaces-delete.html[워크스페이스 삭제 API]
|`DELETE /v1/workspaces/{workspaceId}`
|워크스페이스 OWNER가 워크스페이스와 활성 멤버십을 소프트 삭제하는 요청/응답을 설명합니다.

|link:workspaces-join-requests-invite-code.html[초대 코드 워크스페이스 가입 API]
|`POST /v1/workspaces/join-requests/invite-code`
|초대 코드로 워크스페이스 가입 요청을 만들고 현재 자동 승인 가입 흐름을 완료하는 API를 설명합니다.

|link:workspaces-join-requests.html[워크스페이스 가입 요청 이력 API]
|`GET /v1/workspaces/{workspaceId}/join-requests`
|OWNER 또는 ADMIN이 워크스페이스 가입 요청과 상태 변경 이력을 조회하는 API를 설명합니다.

|link:workspaces-members.html[워크스페이스 멤버 목록 조회 API]
|`GET /v1/workspaces/{workspaceId}/members`
|워크스페이스 활성 멤버 목록, 역할 정렬, 초대 코드 응답을 설명합니다.

|link:workspaces-member-role-update.html[워크스페이스 멤버 역할 변경 API]
|`PATCH /v1/workspaces/{workspaceId}/members/{memberId}/role`
|워크스페이스 OWNER 또는 ADMIN이 대상 멤버의 역할을 변경하는 요청/응답, ADMIN의 OWNER 부여 제한, OWNER 최소 인원 제한을 설명합니다.

|link:workspaces-member-remove.html[워크스페이스 멤버 강제 퇴장 API]
|`DELETE /v1/workspaces/{workspaceId}/members/{memberId}`
|워크스페이스 OWNER 또는 ADMIN이 대상 멤버를 강제 퇴장시키는 요청/응답과 권한 제한을 설명합니다.
|===
