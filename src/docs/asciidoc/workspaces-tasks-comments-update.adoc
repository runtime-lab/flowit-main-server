== 작업 댓글 수정

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``PATCH``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}/comments/{commentId}``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |작업 댓글의 본문을 수정합니다.
|인증 |JWT access token이 필요합니다.
|권한 |댓글 작성자 본인만 수정할 수 있습니다.
|수정 표시 |내용이 실제로 변경되면 이후 목록 조회 응답에서 ``edited``가 ``true``입니다.
|최근 활동 |댓글 수정은 현재 워크스페이스 최근 활동에 기록하지 않습니다.
|성공 응답 |``200 OK``와 빈 성공 응답 본문을 반환합니다.
|===
