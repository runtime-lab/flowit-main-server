== 작업 생성

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``POST``
|경로 |``/v1/workspaces/{workspaceId}/tasks``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |워크스페이스 안에 새 작업을 생성합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|상태 |생성 시 ``status``를 생략하면 ``TODO``로 처리합니다.
|담당자 |``assigneeMemberId``는 ``null``일 수 있으며, 이 경우 미할당 작업으로 생성합니다.
|진행도 |``progress``는 선택 필드입니다. 생략하거나 ``null``이면 ``0``으로 처리합니다.
|태그 |최대 10개까지 허용하며, 같은 작업 안에서는 정규화된 이름 기준으로 중복 제거됩니다.
|설명 |``descriptionMarkdown`` 원문 문자열을 저장합니다. 서버는 마크다운을 렌더링하지 않습니다.
|응답 |``201 Created``
|===
