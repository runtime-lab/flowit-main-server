== 워크스페이스 생성

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``POST``
|경로 |``/v1/workspaces``
|API 상태 |[.badge.stability.stable]#Stable#
|목적 |인증된 사용자가 새 워크스페이스를 생성합니다.
|인증 |JWT access token이 필요합니다.
|생성 결과 |``201 Created``와 ``Location: /v1/workspaces/{id}``를 반환하고, 응답 본문 ``data.createdId``에 생성된 워크스페이스 ID를 제공합니다.
|필드 제한 |``name``은 필수이며 최대 100자입니다. ``description``은 선택이며 최대 500자입니다.
|===
