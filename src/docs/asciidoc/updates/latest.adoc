Integration Note

Docs:: ``Enum Reference`` 문서를 각 도메인에 맞게 분리했습니다.
+
* 도메인 별 Enum 값을 더 쉽게 식별할 수 있도록 문서를 나누고 경량화했습니다.
Docs:: ``Activity``와 ``Notification``문서에 각 도메인의 명시적인 설명을 추가했습니다.
+
* ``Activity``는 사이트 내에서 발생한 활동을 원장 개념으로 기록하는 아카이브입니다.
* ``Notification``은 단발성 이벤트에 대해 사용자에게 알림을 전송했거나 전송하려 한 사실을 기록합니다.
