Integration Note

Empty:: API 변경이 없습니다. 직전 내용을 유지합니다.
Before:: ``POST /v1/workspaces/{workspaceId}/tasks`` 워크스페이스 작업 생성 API가 추가되었습니다.
Before:: ``GET /v1/workspaces/{workspaceId}/tasks`` 워크스페이스 작업 목록 조회 API가 추가되었습니다.
Before:: ``GET /v1/workspaces/{workspaceId}/tasks/{taskId}`` 워크스페이스 작업 상세 조회 API가 추가되었습니다.
Before:: ``PATCH /v1/workspaces/{workspaceId}/tasks/{taskId}`` 워크스페이스 작업 수정 API가 추가되었습니다.
Before:: ``PATCH /v1/workspaces/{workspaceId}/tasks/{taskId}/progress`` 워크스페이스 작업 진행도 수정 API가 추가되었습니다.
Before:: ``GET /v1/workspaces/{workspaceId}/tasks/{taskId}/histories`` 워크스페이스 작업 변경 이력 조회 API가 추가되었습니다.
Before:: ``GET /v1/workspaces/{workspaceId}/activity-records`` 워크스페이스 최근 활동 조회 API가 추가되었습니다.
Additional:: 작업 설명은 ``descriptionMarkdown`` 원문 문자열로 저장합니다. 서버는 마크다운 문법을 렌더링하지 않으며, 표시와 sanitizing은 클라이언트 렌더링 계층에서 처리해야 합니다.
Additional:: 작업 진행도는 작업 일반 수정 API가 아니라 ``PATCH /v1/workspaces/{workspaceId}/tasks/{taskId}/progress``에서만 변경합니다.
Additional:: 최근 활동 응답의 ``items[]`` 레코드 하나가 기본 목록의 한 줄 렌더링 단위입니다. 서버는 문장을 강제하지 않고 ``actor``, ``target``, ``action``, ``changeCount``, ``changedElements`` 의미 키를 반환합니다.
Additional:: 최근 활동의 ``changes[]``는 단일 ``action``에 딸린 상세 diff 목록입니다. 기본 목록 렌더링이 아니라 펼침 또는 상세 UI에서 사용합니다.
