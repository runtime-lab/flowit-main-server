Integration Note

Added:: ``DELETE /v1/workspaces/{workspaceId}/members/{userId}`` 워크스페이스 멤버 강제 퇴장 API가 추가되었습니다.
Additional:: 강제 퇴장은 요청자가 ``OWNER`` 또는 ``ADMIN``일 때만 허용됩니다. ``OWNER`` 대상 강제 퇴장과 자기 자신 강제 퇴장은 허용되지 않습니다.
