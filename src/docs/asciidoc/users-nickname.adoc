== 현재 사용자 닉네임 변경

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |`PATCH`
|경로 |`/v1/users/me/nickname`
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |인증된 사용자의 닉네임을 변경합니다.
|인증 |JWT access token이 필요합니다.
|대상 사용자 |요청 본문이나 경로에서 사용자 ID를 받지 않고, access token의 subject로만 수정 대상을 결정합니다.
|토큰 영향 |닉네임 변경은 `token_version`을 증가시키지 않습니다. 기존 access token과 refresh token은 유지됩니다.
|===
