== 작업 변경 이력 조회

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``GET``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}/histories``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |특정 작업에 대한 변경 이력만 조회합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|표현 방식 |서버는 문장을 강제하지 않고 수행자, 대상, 액션, 변경 요소를 의미 키로 반환합니다.
|최근 활동과의 차이 |워크스페이스 전체 최근 활동은 워크스페이스 최근 활동 API에서 조회합니다.
|응답 |``200 OK``
|===
