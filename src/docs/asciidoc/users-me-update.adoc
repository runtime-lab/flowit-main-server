== 현재 사용자 수정

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |`PATCH`
|경로 |`/v1/users/me`
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |인증된 사용자의 수정 가능한 정보를 변경합니다.
|인증 |JWT access token이 필요합니다.
|대상 사용자 |요청 본문이나 경로에서 사용자 ID를 받지 않고, access token의 subject로만 수정 대상을 결정합니다.
|현재 지원 필드 |현재는 `nickname` 수정을 지원합니다. 향후 수정 가능한 사용자 정보가 추가될 수 있습니다.
|요청 조건 |최소 하나 이상의 수정 필드를 포함해야 합니다. `nickname` 값이 있으면 공백일 수 없고 최대 100자입니다.
|토큰 영향 |사용자 정보 수정은 `token_version`을 증가시키지 않습니다. 기존 access token과 refresh token은 유지됩니다.
|===
