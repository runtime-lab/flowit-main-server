== 워크스페이스 수정

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``PATCH``
|경로 |``/v1/workspaces/{workspaceId}``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |워크스페이스 이름과 설명처럼 수정 가능한 프로필 필드를 변경합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 ``OWNER`` 또는 ``ADMIN`` 멤버여야 합니다.
|PATCH 규칙 |생략한 필드는 변경하지 않습니다. 빈 ``name``은 거부하고, 빈 ``description``은 설명 제거로 처리합니다.
|성공 응답 |수정된 ``Workspace`` 응답 본문을 반환합니다.
|===
