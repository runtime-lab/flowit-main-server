== 작업 전체 수정

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``PATCH``
|경로 |``/v1/workspaces/{workspaceId}/tasks/{taskId}``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |작업의 제목, 설명, 상태, 담당자, 우선순위, 일정, 태그를 한 번에 수정합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|상태 |``TODO``, ``IN_PROGRESS``, ``DONE`` 간 자유 전이를 허용합니다.
|담당자 |다른 워크스페이스 멤버 또는 미할당 상태로 변경할 수 있습니다.
|태그 |최대 10개까지 허용하며, 같은 작업 안에서는 정규화된 이름 기준으로 중복 제거됩니다.
|단독 변경 |상태만 변경할 때는 작업 상태 변경 API를, 진행도만 변경할 때는 작업 진행도 변경 API를 사용합니다.
|제외 필드 |진행도는 이 API의 수정 대상이 아닙니다.
|성공 응답 |``200 OK``와 빈 성공 응답 본문을 반환합니다.
|===
